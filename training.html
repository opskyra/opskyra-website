  • Create and configure storage accounts
  • Configure network access to storage
  • Generate shared access signatures (SAS)
  • Install and use Azure Storage Explorer
  • Implement Azure storage replication
  • Configure Azure Blob storage and CDN
  • Create and configure Azure File Sync
  • Implement Azure Backup and recovery

Module 3: Deploy and Manage Virtual Machines

  • Configure high availability
  • Configure monitoring, networking, and storage
  • Deploy and configure scale sets
  • Automate deployment with ARM templates
  • Manage Azure VM with PowerShell DSC
  • Configure VM backups and Azure Site Recovery

  • Create and configure VNET peering and VNET-to-VNET
  • Configure private and public IP addresses
  • Create and configure Network Security Groups (NSG)
  • Implement Azure Load Balancer
  • Configure Azure DNS
  • Monitor and troubleshoot virtual networking
  • Integrate on-premises network with Azure VPN
  • Configure ExpressRoute

Module 5: Manage Identities

  • Manage Azure Active Directory (Azure AD)
  • Add custom domains and Azure AD Join
  • Configure self-service password reset

  • Implement Multi-Factor Authentication (MFA)
